# LSVortecProDiag.UI

This project contains the user interface layer of the LS‑VortecProDiag application. It is responsible for rendering screens, dialogs, and controls that allow users to interact with diagnostic functionality. Components in this folder implement the visual layout and presentation logic, and orchestrate interactions between the Core library and Windows-specific features. The UI should provide an intuitive, responsive, and accessible experience across supported platforms.
